AryaOS Site Configuration

Site-wide settings in /etc/aryaos/aryaos-config.txt, inherited by every sensor gateway (per-service overrides live in each tool's own Cockpit page). Changes apply on Save & restart sensors.

TAK destination

Default feeds the local Charontak hub; Charontak forwards to Mesh SA and/or a TAK Server (see Charontak lanes). Direct examples: tls://takserver.example.com:8089, udp+wo://239.2.3.1:6969.

Site-wide TAK TLS certificates

Upload once; every PyTAK gateway inherits PYTAK_TLS_CLIENT_CERT/KEY/CAFILE from the site config. PEM format (for .p12 bundles, convert first: openssl pkcs12 -in bundle.p12 -out client.pem -clcerts -nokeys / -nocerts -nodes).
